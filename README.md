# 🌸 NKH Voyages

> **Explore. Dream. Discover.**

Welcome to **NKH Voyages**, a modern travel website designed to inspire travelers to discover beautiful destinations around the world. The website features breathtaking locations, detailed destination pages, travel packages, photo galleries, customer reviews, and booking forms—all presented with a soft pastel color palette for a relaxing and elegant user experience.

## 🌈 Theme Inspiration

The website combines soft and welcoming colors to create a relaxing travel experience.

- 🌸 Pastel Pink (`#F8C8DC`) for the header and footer
- 💜 Pastel Purple (`#C8A2FF`) for hero titles
- 🌷 Soft Pink (`#F8AFCF`) for section headings
- 💙 Ocean Blue (`#0099CC`) for buttons and highlights
- 💛 Golden Yellow (`#CCB400`) for the booking form button
- 🌼 Light Yellow (`#EDF874`) for hover effects
- 🤍 Light Blue (`#F4F9FF`) as the background

This palette creates a clean, elegant, and vacation-inspired design.

## 🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open **index.html** in your web browser.
4. Click on **Bali**, **Tokyo**, or **Türkiye** to explore their dedicated pages.
5. Browse travel packages and book your dream vacation.
