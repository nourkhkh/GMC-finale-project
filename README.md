# 🌍 Velvet Voyages

> **Explore. Dream. Discover.**

Welcome to **Velvet Voyages**, a modern travel website designed to inspire travelers to discover beautiful destinations around the world. The website features breathtaking locations, travel packages, a photo gallery, customer reviews, and a contact form—all presented with a soft pastel color palette for a relaxing and elegant user experience.

### 🌈 Theme Inspiration

This color palette creates a soft, cheerful travel atmosphere by combining:

- 🌸 **Pastel Pink** (`#F8C8DC`) for the header and footer
- 💜 **Pastel Purple** (`#C8A2FF`) for the main welcome title
- 🌷 **Soft Pink** (`#F8AFCF`) for section headings
- 💙 **Ocean Blue** (`#0099CC`) for buttons and accents
- 💛 **Golden Yellow** (`#CCB400`) for the contact form submit button
- 🌼 **Light Yellow** (`#EDF874`) for hover effects
- 🤍 **Light Blue Background** (`#F4F9FF`) for a fresh, clean look

The combination gives **Velvet Voyages** a bright, elegant, and welcoming design that reflects beaches, sunshine, and unforgettable travel experiences.
